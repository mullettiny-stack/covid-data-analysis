# COVID-19 Data Analysis

## Project Overview

This project analyzes COVID-19 data using Python, Pandas, and Matplotlib. The goal is to clean, explore, and visualize the dataset to uncover trends in confirmed, recovered, and death cases.

## Tools Used

- Python
- Pandas
- Seaborn
- Microsoft Excel
- Matplotlib
- Jupyter Notebook

## Features

- Data cleaning
- Data exploration
- GroupBy analysis
- Bar chart visualizations
- COVID-19 statistics by country and continent
